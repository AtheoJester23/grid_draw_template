# 🎨 GridArt Editor

GridArt Editor is a web-based tool for artists to create **reference grids** for drawings, paintings, or digital art. Easily overlay customizable grids on preset canvas sizes to improve proportions, perspective, and composition.

---

## 🌟 Features

- **Preset canvas sizes**: Choose from **A4**, **6x8 inches**, or **4x6 inches** workspaces.  
- **Custom grid spacing**: Select grid sizes like **1x1 inch**, **2x2 inches**, **3x3 inches**, and more.  
- **Live preview**: See the grid update in real-time before downloading.  
- **Export options**: Download your grid as PNG or SVG for use in digital or traditional art.  
- **User-friendly interface**: Focus on creating art without distractions.

---

## 🌟 Features

- **Customizable grids**: Choose various grids to match your artwork needs.
- **Flexible sizing**: Adjust grid dimensions to any canvas size.
- **Export options**: Export image as PNG or JPG to use in any digital art software.
- **User-friendly interface**: Intuitive design so artists can focus on their creativity.
- **Live preview**: See your grid update in real-time as you make adjustments.

---

## 🖌️ How to Use

1. Open [GridArt Editor](#) in your browser.  
2. Select your **canvas size**: A4, 6x8, or 4x6 inches.  
3. Choose your desired **grid size** (e.g., 1x1 inch, 2x2 inches, etc.).  
4. Preview your grid in real-time.  
5. Download the grid and use it as a reference layer in your artwork.
